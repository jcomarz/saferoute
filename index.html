SafeRoute

Finds the driving route with the fewest tricky intersections — penalising unprotected left turns (turns across oncoming traffic) and U-turns, and rewarding turns that happen at traffic signals.

Needs a key with Maps JavaScript API, Directions API and Places API (New) enabled (billing on). Stored only in this browser (localStorage), never committed or sent anywhere but Google.
Get one at Google Cloud Console.

Traffic-signal data comes from OpenStreetMap (Overpass API) — free, no key.
